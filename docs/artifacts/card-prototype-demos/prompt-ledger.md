# Card Prototype Prompt Ledger

These prompts produced the approved Round 2 card-art pitch sheets. Keep them as
the baseline for future 60-card production prompts.

## Prototype 1: Type Grammar, Round 2

```text
Use case: stylized-concept
Asset type: Spirebound card-art prototype demo sheet, preview only

Primary request:
Create a bright five-tile contact sheet for Spirebound card art type grammar. Each tile shows one centred stained-glass emblem on a warm off-white parchment review background, with plenty of negative space. This is not a final card, not UI, not a card frame, not text.

Game/style context:
Spirebound is a browser roguelite deckbuilder with serious cartoon-gothic stained-glass art. The generated art must be readable inside a tiny 64 px card-art window. Use chunky dark lead outlines, large simple jewel-glass shapes, matte painterly glass, warm amber rim light, transparent-looking object edges, and high-value colours. Do not make the image dark overall. Do not use black-on-black detail.

Five tiles, left to right:
1. Attack / Edge: crimson stained-glass blade slash, diagonal cut, red shards, sharp silhouette.
2. Skill / Ward: blue-cyan shield pane with a small amber lantern centre, calm enclosing shape.
3. Power / Vow: violet and gold rose-window heart flame, symmetrical persistent aura, ceremonial halo.
4. Status / Ash: ash-grey cracked card shard with sick green ember smoke, harsh dead-weight object.
5. Curse / Hex: void-purple warped hex sigil trapped in black lead geometry, toxic magenta inner glow.

Rarity and type signalling:
This sheet is about type grammar only. Keep each tile equal in power and size. The five types must be obviously distinct by silhouette, not only colour.

Lighting:
Bright review sheet, clear silhouettes, luminous glass, no murky background. Target a lighter, clearer second-round correction.

Negative constraints:
No words, no labels, no numbers, no UI frame, no cost badge, no card border, no photorealism, no realistic gore, no tiny details, no full characters, no dark cave background.
```

## Prototype 2: Rarity Ladder, Round 2

```text
Use case: stylized-concept
Asset type: Spirebound rarity ladder prototype sheet, preview only

Primary request:
Create a bright five-tile contact sheet showing the same Ward / defence card idea across Spirebound rarity levels. The goal is to prove rarity is passed visually, not by labels or text. Each tile should be a centred stained-glass shield/lantern emblem on a warm off-white parchment review background. No words, no labels.

Game/style context:
Serious cartoon-gothic stained-glass game art, chunky dark lead outlines, large clear jewel panes, matte painterly texture, warm amber rim light, high-value colours. The image must be readable at 64 px. Overall brightness should be clear and lifted, not dark.

Five tiles, left to right, with escalating rarity:
1. Starter: very simple blue shield pane with a tiny amber lantern dot, plain silhouette, one object.
2. Common: blue shield with one silver mirrored arc and a stronger lantern glow, still simple.
3. Uncommon: shield plus two side facet shards, clearer cyan highlights, more asymmetry, one combo cue.
4. Rare: cathedral rose-window shield, gold lead accents, strong halo, ceremonial but still one main silhouette.
5. Rare Power variant: shield becomes persistent violet-gold vow halo around a lantern flame, highest ceremony, more radiant but not cluttered.

Rarity must pass through:
- silhouette complexity
- amount of gold/silver material
- strength of halo/light
- architectural ceremony
- supporting facet count

Keep all tiles same object family so progression is visible. Avoid making them repeated copies.

Negative constraints:
No text, no labels, no card frame, no UI chrome, no character portrait, no tiny runes, no black background, no murky low-key lighting.
```

## Prototype 3: Signature Mechanics, Round 2

```text
Use case: stylized-concept
Asset type: Spirebound signature mechanics prototype sheet, preview only

Primary request:
Create a bright six-tile contact sheet of Spirebound's signature mechanic symbols. Each tile is one large centred stained-glass emblem on a warm off-white parchment review background. No text or labels. These are pitch-sheet symbols, not production assets.

Game/style context:
Serious cartoon-gothic stained-glass game art. Use chunky dark lead outlines, large readable jewel panes, matte painterly glass texture, warm amber rim light, clear silhouettes, and lifted brightness. Must work as a tiny card-art read at 64 px.

Six tiles, left to right:
1. Facet chip: a blue crystal pane with one corner cleanly chiselled away, visible missing shard.
2. Cracked: a pale glass shield split by one strong fracture line, not shattered into noise.
3. Kindle / draw: a small card pane whose lower edge burns inward into an amber ember while a second card silhouette is pulled upward into light.
4. Ember economy: a lantern bowl holding three distinct amber coal beads, simple and countable.
5. Smolder attack: green-orange ember bite mark cutting into a red glass edge, smoky but readable.
6. Smolder catalyst: an ash cloud spiral feeding a brighter coal, controlled green-orange smoke, combo feeling.

Lighting:
Bright and clean second-round correction. Avoid black-on-black details. Keep symbols large, simple, and separated.

Negative constraints:
No words, no labels, no numbers, no UI/card frame, no realistic gore, no full characters, no dark cave or black background, no micro-detail.
```
