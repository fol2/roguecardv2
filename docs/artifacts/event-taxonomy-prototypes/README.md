## Decision Record

Round 1 failed because the event scenes were too dark.

Round 2 corrected brightness, but the event images still felt too fragile:
background windows, floor grids, armour segments, and fine lead lines competed
with the event focal read.

Round 3 improved the main subjects by borrowing the successful hero/enemy
stroke language: chunkier silhouettes, fewer interior dividers, and simpler
colour masses. It still leaned too far towards isolated object clusters.

Round 4 is the approved event-art direction: keep the strong chunky subject, but
add a narrative background with controlled depth. Use foreground subject,
midground story prop, and broad low-detail background shapes. The background
must have lower contrast and fewer lines than the subject.

The prompts used for the approved Round 4 set are stored in
[`prompt-ledger.md`](./prompt-ledger.md).
