# GPT -> Nano Pass Card Comparison 03

This scratch set contains cards 11-20 in gallery/data order for the
`gpt-image-2 -> Nano Banana Pro` image-to-image comparison set. `venomStrike`
and `leechBlade` were regenerated by user request with the same dedicated
prompt.

Cards generated:

- `cleave` / Fan of Glass
- `venomStrike` / Emberbite
- `lunge` / Dimming Cut
- `guardedStrike` / Warden's Edge
- `brace` / Held Light
- `sidestep` / Glasstep
- `preparation` / Tinder
- `deflect` / Refract
- `leechBlade` / Thirsting Shard
- `tempest` / Hailglass

Workflow:

1. Used the existing live GPT card art from `src/assets/cards/<id>.png` as the
   Nano Banana Pro input image.
2. Used the short Nano Banana pass prompt plus style frame only; no stored GPT
   source prompt was appended.
3. Generated with
   `/Users/jamesto/.codex/skills/nanobanana/scripts/nanobanana.py`.
4. Copied accepted source bytes directly to `src/assets-gpt-nano-pass/cards/`.
5. Reviewed through `?gallery=1&set=gpt-nano-pass`.

No accepted source image was locally post-processed. The contact sheet
`live-vs-gpt-nano-pass-contact-03.jpg` is review-only; it is not a source asset.

Accepted source-to-gallery byte checks:


used the same prompt, but drifted into a clean character-illustration read
instead of preserving the live card composition.

`venomStrike-gpt-nano-pass-v1-superseded-by-redo.jpg` and
`leechBlade-gpt-nano-pass-v1-superseded-by-redo.jpg` are kept as superseded
