# Event Taxonomy Prototype Prompt Ledger

Prototype set for pitching Spirebound event categorisation and visual treatment.
These are built-in Image Gen pitch demos, not promoted runtime assets.

Contact sheet: `contact-sheet.jpg`

## Taxonomy Proposal

- Event type: `sanctuary`, `bargain`, `craft`, `danger`, `encounter`.
- Risk tier: `safe`, `priced`, `gamble`, `cursed`, `omen`.
- Rarity: `common`, `uncommon`, `rare`, `mythic`.
- Visual role: `object`, `place`, `figure`, `ritual`.
