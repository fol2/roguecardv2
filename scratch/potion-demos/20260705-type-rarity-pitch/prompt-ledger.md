# Potion Type/Rarity Pitch Demos

Purpose: quick visual pitch boards for potion taxonomy, rarity language, and a
rare mini-set. These are concept demos, not final alpha-ready runtime assets.

Workflow: built-in Codex Image Gen, no CLI fallback, no Nano Banana pass, no
alpha cutout.
