# Legacy Asset Set

Previous live runtime raster set, demoted when the readable baseline was
promoted to `src/assets/` on 2026-07-06.

This folder mostly mirrors `src/assets/<category>/<id>.png` and is registered as
the `legacy` gallery set in `src/art.js` for visual comparison. It intentionally
does not include the newer `title/title.png` wordmark, because that asset did
not exist in the old live set. The old title-screen background is archived as
`title-background/background.png`, leaving legacy `title` blank. Do not add new
production art here unless restoring or archiving the old live set.
