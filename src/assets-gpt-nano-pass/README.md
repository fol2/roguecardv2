# GPT -> Nano Pass Asset Set

Comparison-only raster set for reviewing the current live GPT-generated card art
after an image-to-image Nano Banana Pro clean-up pass.

Normal gameplay still uses the `live` set in `src/assets/`. This folder is
registered as the `gpt-nano-pass` gallery set in `src/art.js` so it can be
opened with:

```text
http://localhost:5174/?gallery=1&set=gpt-nano-pass
```
